# Secure-Random-Password-Generator

SecureGen Pro is a modern password and security key generator built with HTML, CSS, and JavaScript. It focuses on strong key creation, real-time strength analysis, and an intuitive user experience. The tool includes history tracking, theme switching, a security audit system, and an optional auto-refresh mode for temporary keys.

## Key Features

### Strong Key Generation
- Uses `crypto.getRandomValues()` for high-entropy keys  
- Customizable character sets (uppercase, lowercase, numbers, symbols)  
- Option to remove ambiguous characters for better readability  

### Real-Time Strength Analysis
- Live entropy calculation in bits  
- Visual strength meter  
- Detection for patterns, repeated characters, and sequences  
- Special character count and password age tracking  

### Security Tools
- One-click Security Audit with pass/fail criteria  
- Auto-refresh for temporary passwords (30 seconds, 1 minute, 5 minutes)  

### Password History
- Stores the last 20 generated keys locally  
- Export history as a text file  
- One-click delete history  

### UI and UX
- Minimal and modern UI with smooth transitions  
- Light and dark theme support  
- Scramble animation during key generation  
- Clipboard copy button and visibility toggle
