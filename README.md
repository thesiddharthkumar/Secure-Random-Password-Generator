# Secure-Random-Password-Generator

SecureGen Pro is a modern security key and password generator built with pure HTML, CSS, and JavaScript. It focuses on strong password creation, real-time strength analysis, and a smooth user experience. It also includes history tracking, theme switching, a security audit tool, and optional auto-refresh for temporary keys.

## Key Features

### Strong Key Generation
- High-entropy random generation using `crypto.getRandomValues()`.
- Fully customizable character sets: uppercase, lowercase, numbers, symbols.
- Options to avoid ambiguous or similar-looking characters.

### Real-Time Strength Analysis
- Live entropy calculation (bits).
- Visual strength meter.
- Pattern detection for repeated characters or sequences.
- Special character counter and password age tracking.

### Security Tools
- One-click Security Audit showing pass/fail on key criteria.
- Auto-refresh timer for temporary passwords (30s, 1 min, 5 min).

### Password History
- Automatic local history storage (up to 20 recent keys).
- Export history as a text file.
- Delete history with one click.

### UI & UX Highlights
- Clean, modern interface using CSS animations and smooth transitions.
- Light and dark theme toggle.
- Scramble animation when generating new keys.
- Clipboard copy button and visibility toggle.
